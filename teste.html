Import Sales Data

Upload your Excel or CSV file to seamlessly import sales records.

Instructions

  1. Select the file type you want to import (Excel or CSV).
  2. Click the "Browse Files" button or drag and drop your file into the designated area.
  3. Ensure your file is no larger than 10MB.
  4. Once the file is selected, click "Upload File" to begin the import process.
  5. You will be redirected to a review page to confirm the data before finalizing.

File Format

Please ensure your file has the following columns in order:

  • Sale ID
  • Date
  • Product Name
  • Amount
  • Architect Name
File Type

XLS, XLSX, or CSV up to 10MB
